# 学情智能分析与预警系统 — 演示录制脚本

> 适用：创 AI 案例征集活动演示视频  
> 建议总时长：6~8 分钟  
> 结构：PPT 封面（30s）+ 系统录屏演示（5~6min）+ 结尾总结（30s）

---

## 一、PPT 封面页（30 秒）

**画面**：静态 PPT，展示项目基本信息  
**解说词**：
> "大家好，我是 XXX，今天展示的项目是《学情智能分析与预警系统》。这是一款面向基础教育教师的 Web 应用，帮助教师通过上传成绩 Excel，自动完成学情统计、薄弱点识别、风险预警，并生成可下载的 AI 分析报告。项目采用 React + FastAPI 全栈自研，未使用低代码平台。"

**PPT 内容要点**：
- 项目名称 + 团队/个人
- 一句话价值主张
- 技术栈图标（React、FastAPI、SQLite、ECharts、DeepSeek）

---

## 二、系统演示 — 按以下顺序操作

### 步骤 1：登录（20 秒）

**操作**：
1. 打开浏览器，访问 `http://localhost:5173`（或部署地址）
2. 输入账号 `T2024001`，密码 `admin`
3. 点击登录

**解说词**：
> "系统采用 JWT 认证，内置默认教师账号，开箱即用。"

**检查点**：
- [ ] 登录成功跳转班级列表页
- [ ] 左侧菜单栏完整显示

---

### 步骤 2：班级管理（30 秒）

**操作**：
1. 展示班级列表页（默认进入）
2. 点击「新建班级」按钮（如有）或展示已有班级
3. 展示班级信息：名称、学科、年级、学校、学生数

**解说词**：
> "教师首先创建或选择班级。系统支持多班级管理，每个班级独立管理学生、考试和分析数据。"

**检查点**：
- [ ] 班级列表正常展示
- [ ] 操作按钮可见：仪表盘、导入成绩、学生画像、风险预警

---

### 步骤 3：Excel 成绩导入（45 秒）

**操作**：
1. 在班级列表中点击「导入成绩」
2. 填写考试名称（如"期中考试"）和日期
3. 选择 `一年级.xlsx` 模板文件上传
4. 点击「开始导入」
5. 等待导入成功提示

**解说词**：
> "教师上传成绩 Excel，系统按模板自动解析，建立考试记录、学生档案和成绩数据。这里我们导入一次期中考试成绩。"

**检查点**：
- [ ] 导入成功提示显示导入条数
- [ ] 提示中显示具体导入人数

---

### 步骤 4：学情仪表盘（60 秒）

**操作**：
1. 从班级列表点击「仪表盘」
2. 展示 KPI 卡片：班级均分、标准差、及格率、优秀率
3. 展示分数段分布柱状图
4. 向下滚动展示学生成绩明细表格
5. 点击表头「总分」排序，展示排序功能

**解说词**：
> "导入完成后，系统自动计算统计指标。仪表盘展示班级均分、标准差、及格率和优秀率四个核心指标。下方是分数段分布图，直观呈现成绩结构。底部是每位学生的成绩明细，支持排序。"

**检查点**：
- [ ] 4 个 KPI 卡片数据正常
- [ ] 柱状图渲染正常
- [ ] 表格可排序

---

### 步骤 5：学生画像（60 秒）

**操作**：
1. 从班级列表点击「学生画像」
2. 展示左栏学生列表
3. 点击第一名学生，展示右侧画像
4. 展示：基本信息、薄弱学科标签（红色）
5. 展示「成绩趋势」折线图
6. 展示「学科雷达」图
7. 换另一名有薄弱学科的学生，展示对比

**解说词**：
> "学生画像模块支持教师查看每位学生的学习轨迹。左侧是班级学生列表，点击后右侧展示该生的成绩趋势折线图和学科雷达图，系统自动标记得分低于 70 分的薄弱学科，帮助教师精准定位需要重点关注的学生。"

**检查点**：
- [ ] 左右分栏布局正常
- [ ] 点击学生后右侧数据刷新
- [ ] 折线图和雷达图正常渲染
- [ ] 薄弱学科标签显示正确

---

### 步骤 6：知识点热力图（45 秒）

**操作**：
1. 点击左侧菜单「知识点热力图」
2. 选择考试（如"期中考试"）
3. 展示热力图：X 轴知识点、Y 轴学生、颜色深浅映射分数
4. 鼠标悬停展示具体分数

**解说词**：
> "知识点热力图将学生成绩按知识点维度展开。横轴是知识点，纵轴是学生，颜色越深代表掌握越好。教师一眼就能看出全班在哪个知识点上存在共性问题，便于针对性备课。"

**检查点**：
- [ ] 热力图渲染正常
- [ ] 颜色映射正常
- [ ] 悬停 tooltip 显示学生+知识点+分数

---

### 步骤 7：风险预警（60 秒）

**操作**：
1. 点击左侧菜单「风险预警」（或从班级列表点击「风险预警」按钮）
2. 选择考试
3. 展示三个统计卡片：高风险、中风险、低风险人数
4. 向下滚动展示风险学生列表
5. 展示风险等级标签（红色=高风险）、风险类型、预警原因、建议措施

**解说词**：
> "系统自动检测三类风险：连续下滑、大幅退步和偏科严重。这里可以看到高风险和中风险学生的数量统计。下方的列表详细列出每位风险学生的预警原因和针对性的教学建议，比如'关注学习状态，排查外部干扰因素'，帮助教师提前干预。"

**检查点**：
- [ ] 三个统计卡片数据正常
- [ ] 风险学生表格渲染正常
- [ ] 高风险标签为红色
- [ ] 预警原因和建议文本完整

---

### 步骤 8：AI 分析报告（60 秒）

**操作**：
1. 回到学情仪表盘页面
2. 点击顶部「生成 AI 报告」按钮
3. 等待报告生成（展示 loading）
4. 展示生成的报告内容（Markdown 格式）
5. 滚动浏览报告：班级概况、成绩分析、风险分析、教学建议

**解说词**：
> "系统集成了 DeepSeek AI 大模型，基于统计数据、风险预警结果和知识点分布，自动生成学情分析报告。报告包含班级概况、成绩分布分析、风险学生分析和针对性的教学改进建议，帮助教师从数据中发现教学问题。"

**检查点**：
- [ ] 点击按钮后跳转报告页
- [ ] Loading 状态正常
- [ ] 报告内容包含真实数据（非 Lorem ipsum）
- [ ] 标题、段落、列表格式正常

---

### 步骤 9：报告导出（30 秒）

**操作**：
1. 在报告页面点击「导出 Word」按钮
2. 浏览器下载 docx 文件
3. 点击「导出 PDF」按钮
4. 浏览器下载 pdf 文件
5. （可选）快速展示下载的文件图标

**解说词**：
> "生成的报告支持一键导出为 Word 和 PDF 格式，方便教师存档、打印或提交给学校教务处。"

**检查点**：
- [ ] Word 导出按钮可点击
- [ ] PDF 导出按钮可点击
- [ ] 文件正常下载

---

## 四、录制技术建议

| 项目 | 建议 |
|------|------|
| 录屏工具 | OBS / Camtasia / 剪映专业版 |
| 分辨率 | 1920×1080 |
| 鼠标高亮 | 开启，方便评委跟随操作 |
| 旁白录制 | 先写稿后录音，避免口头禅 |
| BGM | 不加，保持专业感 |
| 字幕 | 关键页面可配简短字幕说明 |

## 五、预演检查清单

录制前请确认：

- [ ] 后端服务已启动 (`uvicorn app.main:app --port 8001`)
- [ ] 前端服务已启动 (`npm run dev`)
- [ ] 数据库已初始化且有演示数据（至少 1 个班级、1 次考试、30+ 学生）
- [ ] 已导入 `一年级.xlsx` 模板数据
- [ ] 风险预警页面有风险学生数据（如没有，需提前构造测试数据）
- [ ] AI 报告生成正常（或 fallback 模板正常）
- [ ] 导出 Word/PDF 功能正常
- [ ] 浏览器缓存已清除（避免旧代码干扰）
- [ ] 录屏软件测试通过
- [ ] 解说稿已熟读 2 遍
